Fracturebound Mobile RPG - Publish Ready

This folder is ready to upload to GitHub Pages or itch.io.

INCLUDED
- index.html  -> the whole game in one file
- .nojekyll   -> helps GitHub Pages serve the site cleanly

GITHUB PAGES
1. Create a new GitHub repository.
2. Upload index.html and .nojekyll to the root of the repo.
3. Open the repo Settings.
4. Go to Pages.
5. Under Build and deployment, choose:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: / (root)
6. Save.
7. GitHub will give you a public URL like:
   https://yourusername.github.io/repository-name/

ITCH.IO
Option 1: Quick browser upload
1. Log into itch.io
2. Create a new project
3. Choose "HTML"
4. Upload this folder as a ZIP
5. In the embed options, set the game size to something like 720 x 1280 or responsive
6. Save and view the project page

Option 2: ZIP upload
- Upload the included ZIP file directly if itch.io asks for a downloadable upload

NOTES
